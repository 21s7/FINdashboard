findashboard — это веб-приложение для создания и визуализации инвестиционных портфелей. Пользователи могут комбинировать активы (акции, криптовалюты, облигации, драгметаллы и др.) в одном портфеле, просматривать аналитику и делиться результатами этого портфеля. 

Ключевые функции:
-Создание портфеля – выбор активов акции облигации драгметалы крипта, валюты просмотр
-Просмотр статистики собранного из активов портфеля общая ценность портфеля 
-Анимированные графики – визуализация структуры портфеля
-Экспорт в PDF/PNG – красивый скриншот для соцсетей
-Поделиться ссылкой – URL с параметрами (например, ??.com/?assets=btc50,gold30)
-Тёмная тема – удобный интерфейс для финансовых данных

Технологический стек
Frontend React (js) 
Стили SCSS (модули)  
API выделил в слайсы 


Короче findashboard – это удобный инструмент для визуализации инвестиционных портфелей с возможностью экспорта и обмена.
